What is rawfrontend?

This is a front-end project that intends to put into practice some of the knowledge I've been acquiring so far, since I've started my journey on the web a few months ago.

For now, it is a work in progress that will be exploring the development process of this website through some of the HTML, CSS and JavaScript code that I think are worth highlighting. As the content grows, new features and better solutions might be added/implemented. At this time, for example, the website isn't fully optimized for the desktop (only mobile).

By sharing and trying to explain (in a casual but accurate way) those bits of code, I hope this project serves as a practical example of what can be done with those 3 front-end core technologies. So, especially for web developers who are also at the beginning of this long road, it may provide some useful hints and new ideas that can help to solve similar learning challenges.

To complement this project and in addition to GitHub (where it is possible to check all files and code changes) there is also a Twitter account where I'll be posting the latest articles. It is an easy way to keep track of them as they are released and, most importantly, it's a good place to discuss different perspectives and share related approaches/ideas. I'd really appreciate to receive all kind of feedback, because in the end we all want to learn more and keep improving our developer skills!

— Pedro F. Soares
